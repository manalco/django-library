# Used Tecnologies

* Django
* djangorestframework
* djangorestframework_jwt
* django-cors-headers
* Vue.js
* Bootstrap


---


# To install

1. `python manage.py makemigrations; python manage.py migrate`
2. `python manage.py loaddata fixtures/all.json` Optional to load predefined data
3. `cd frontend; npm install`
